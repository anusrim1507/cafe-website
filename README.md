# ☕ Sunrise Café Website

A modern and responsive café website built using Flask.
This project simulates a real-world food ordering system with a clean UI and backend logic.

---

## 🚀 Features

* 🖥️ Beautiful homepage with menu display
* 🛒 Add to Cart functionality
* 💰 Dynamic price calculation
* 📦 Order placement system
* 🧑‍💼 Admin panel to view orders
* 📱 Responsive design

---

## 🛠️ Tech Stack

* Python
* Flask
* HTML, CSS
* Jinja Templates

---

## ▶️ How to Run

1. Install Flask:

   ```bash
   pip install flask
   ```

2. Run the app:

   ```bash
   python app.py
   ```

3. Open in browser:

   ```
   http://127.0.0.1:5000/
   ```
